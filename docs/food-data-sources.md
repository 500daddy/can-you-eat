# 食材库数据来源与安全边界

更新日期：2026-06-25

## 数据定位

“宝宝食材小管家”的食材库用于记录食材、辅助筛选分类、提供保存提醒和宝宝食用建议参考。它不能替代专业医疗或食品安全判断，也不能替代医生、注册营养师、食品安全监管机构或产品包装说明。

## 权威来源优先级

1. 国家卫生健康委：婴幼儿喂养、托育机构营养与安全原则。
2. 中国疾病预防控制中心：婴幼儿辅食喂养、食品安全与营养科普。
3. 中国营养学会：中国居民膳食指南、中国婴幼儿喂养指南和膳食分类。
4. 中国疾控中心营养与健康所 / 中国营养学会：食物营养成分查询平台，用于补充原料类目。
5. FoodSafety.gov 等官方食品安全资料：用于冷藏、冷冻保存边界参考。

## 分类规则

一级分类服务于家长快速筛选，保持少而清晰：

- 蔬菜
- 水果
- 肉禽水产
- 蛋奶豆制品
- 主食辅食

二级分类用于细分食材来源和购物习惯：

- 蔬菜：叶花菜类、根茎薯芋类、茄果瓜类、鲜豆类、菌藻类、葱蒜类。
- 水果：仁果类、核果类、浆果类、柑橘类、瓜果类、热带水果。
- 肉禽水产：畜肉类、禽肉类、水产类。
- 蛋奶豆制品：蛋类、奶制品、豆制品。
- 主食辅食：谷物杂粮、面点粉类、熟食、辅食谷粉、辅食泥。

“根茎类”“菌藻类”等不作为一级分类，因为它们属于蔬菜内部的二级筛选；“蛋白”不作为分类，因为它是营养素视角，不利于用户按真实食材采购和记录。

## 婴幼儿安全表达

食材条目中的宝宝建议使用保守措辞：

- 新食材应一次一种、少量尝试，并观察不适反应。
- 肉、禽、水产、蛋、豆制品和熟食给宝宝前应充分加热。
- 水果、蔬菜需清洗；葡萄、樱桃、坚果粒、毛豆等存在呛噎风险的食材必须按月龄处理形态。
- 1 岁以内辅食保持原味，不额外添加盐、糖和刺激性调味品。
- 出现霉变、酸败味、发黏、胀包、异常出水、明显变色等情况时，不建议继续给宝宝食用。

## 保存提醒边界

本项目的“宝宝建议期”和“成人参考期”是保守提醒，不是绝对安全期限。实际是否可食用仍需结合：

- 购买时新鲜度和包装状态。
- 是否全程冷链或低温保存。
- 是否清洗、切开、烹饪或开封。
- 家庭冰箱温度和交叉污染风险。
- 宝宝月龄、过敏史、身体状态和医生建议。

冷冻保存时间主要影响品质；持续合格低温下不等于任何食材都适合直接给宝宝食用。解冻后应尽快充分加热，避免反复解冻。
